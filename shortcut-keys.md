 # pycharm常用快捷键

|             |                            |
| ----------- | -------------------------- |
| shift + F10 | 运行                       |
| alt + enter | 导包                       |
|             |                            |
| ctrl + g    | 跳转行                     |
| ctrl + home | 跳转开头                   |
| ctrl + end  | 跳转结尾                   |
| ctrl + d    | 复制该行插入到下一行       |
| ctrl + y    | 删除当前行                 |
| ctrl + p    | 显示函数参数的说明         |
|             |                            |
| alt  + j*   | 选中当前变量               |
| shift + F4  | 选中当前变量的所有同名变量 |
